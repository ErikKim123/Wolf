# JNJ SCORE — Web UI Kit

Pixel-fidelity recreation of the JNJ SCORE web product surface using the Nike-inspired design system. Components are intentionally cosmetic — they look right, but state and data are mocked.

**Open `index.html`** to see the interactive demo: home → live scoreboard → shop → product detail.

## Components

- `Nav.jsx` — promo banner + sticky white top nav with logo, category links, search, favorites, bag
- `Hero.jsx` — full-bleed photographic hero with massive uppercase headline + pill CTA
- `Scoreboard.jsx` — live match strip (dark variant) with LIVE indicator + match details
- `ProductGrid.jsx` — tight-gap 3-column product grid
- `ProductCard.jsx` — image + title + meta + price (no shadow, no radius on image)
- `FilterBar.jsx` — pill-shaped category filters
- `Footer.jsx` — dark inverted footer with link columns
- `Button.jsx` — Primary / Secondary / Inverse with sm sizing
- `Badge.jsx` — Sale / Live / New badge variants
